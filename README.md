# React-Native 앱을 이용한 날씨 앱 UI 수정 과제

## How to run

### Clone the repository

```bash
$ git clone https://github.com/cwdoh/RN101-class.git
```

### Install whole dependencies

```bash
$ cd RN101-class
$ npm install
```

### Execute

```bash
$ npm run start

```
